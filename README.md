# A.R.D. Digital Catalog v0.3

Versione pronta per GitHub Pages.

## Pubblicazione rapida
1. Crea un repository GitHub, ad esempio `ARD-Digital-Catalog`.
2. Carica tutto il contenuto di questa cartella, non la cartella esterna.
3. Vai su Settings > Pages.
4. Source: Deploy from branch. Branch: main, folder: /root.
5. Salva e attendi 1-2 minuti.

## Aggiornamento promozioni
- Aggiungi una nuova immagine in `assets/img` e una miniatura in `assets/thumbs`.
- Aggiorna i dati nel file `data/catalogo.json` oppure rigenera il sito dal file Excel master.

Nota: questa versione incorpora anche i dati direttamente in `index.html`, quindi funziona anche offline.
